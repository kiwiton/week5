with this project i will try to connect the bmp280 sensor & bh1750, read them, and after that we display them on an oled display

the schema sensoreWeek5 shows how I wired thi project together.

in main.ccp you find the code i used its foreseen to be uploaded in the esp32 and comunicates via I2C

Used Components:
  esp32 as microcontroller
  bh1750 (digital lux sensor)
  bmp280 (gigital temperature sensor)
  
pins i used:
  3.3V
  GND
  21 (sda)
  22 (SCL)
